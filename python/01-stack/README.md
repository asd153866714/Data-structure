# Stack (堆疊)

## 中序轉後序

步驟 :

1. 中序表示式加入括號，需考慮運算子的優先順序
2. 運算子移到對應的括號右邊
3. 去掉括號

範例 :

1. a + b // c
2. ((a+b) // c)
3. ((ab) + c ) //
4. ab + c //
